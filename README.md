
# Dart File I/O Demo

โปรเจกต์ตัวอย่างที่แสดงการทำงานของ Dart กับไฟล์ประเภทต่าง ๆ (TXT, JSON และ CSV) ผ่านเมนูแบบ command line ใน `bin/main.dart` เพื่อศึกษาวิธีอ่าน/เขียนไฟล์บนดิสก์

## โครงสร้างโปรเจกต์

- `bin/main.dart` – เมนูหลักให้เลือก demo แต่ละแบบ
- `lib/txt_io.dart` – ตัวอย่างอ่านไฟล์ข้อความธรรมดา
- `lib/json_io.dart` – ตัวอย่างอ่านและอัปเดตไฟล์ JSON (`data/students.json`)
- `lib/csv_io.dart` – ตัวอย่างอ่านไฟล์ CSV (`data/scores.csv`)
- `data/` – โฟลเดอร์เก็บไฟล์ตัวอย่าง (`notes.txt`, `students.json`, `scores.csv`)

## วิธีรัน

```bash
dart run bin/main.dart
```

จากนั้นเลือกเมนู 1-3:
1. **Demo TXT** – อ่าน `data/notes.txt` แล้วพิมพ์เนื้อหา
2. **Demo JSON** – แสดงรายชื่อนักเรียนจาก `students.json`, เพิ่ม `I-tim` แล้วเขียนกลับไฟล์
3. **Demo CSV** – แสดงข้อมูลคะแนนจาก `scores.csv` และสร้างไฟล์ให้ถ้าหายหรือว่าง

> หมายเหตุ: เมนู JSON และ CSV จะสร้างไฟล์ตัวอย่างให้เอง หากไฟล์ไม่พบหรือถูกลบไปก่อนหน้า

## การปรับแต่ง

- อัปเดตข้อมูลใน `data/*.json` หรือ `data/*.csv` แล้วรันซ้ำเพื่อดูผล
- เพิ่มเมนูใหม่ได้โดยสร้างฟังก์ชันใน `lib/` แล้วแก้ `switch` ใน `bin/main.dart`
- ถ้าไม่ต้องการให้ JSON เพิ่ม `I-tim` ซ้ำ ให้เช็กว่ามีอยู่แล้วก่อน `students.add(...)`

## ความต้องการระบบ

- Dart SDK 3.x (ตรวจสอบด้วย `dart --version`)

โปรเจกต์นี้เหมาะสำหรับศึกษาการจัดการไฟล์พื้นฐานด้วย Dart และสามารถต่อยอดเป็นยูทิลิตีหรือ CLI ที่ซับซ้อนขึ้นได้ตามต้องการ
